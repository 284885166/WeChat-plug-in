微信6.6.1 xposed模块
====

Version update
====
增加了朋友圈转发。群发助手，修改了红包领取模式，增加了红包 自动回复个性化设置。里面还有隐藏功能。一键添加群好友。修复了一些问题

https://github.com/skyun1314/WeChat-plug-in/releases/download/1.5/wx_plug_in3-release.apk

function
==== 

 * 1自动回复(可设置指定人指定群自定义模式) 
 * 2防撤回  
 * 3抢红包(可设置指定人指定群自定义模式) 
 * 4骰子作弊 
 * 5模拟位置  
 * 6步数最高   
 * 7微信跳一跳助手
 * 8朋友圈转发
 * 9群发助手(可群发文字，图片，视频，语音，收藏，名片) 
 * 10还有一个隐藏功能一键添加群好友，删除群好友
